del (Deleted Text) This element is used to indicate that text has been deleted from a document. A browser might render deleted text as strikethrough text.
ins (Inserted Text) This element is used to indicate that text has been added to the document. Inserted text is generally styled with an underline.
u (Underline) This element indicates that the enclosed text should be displayed underlined. It is deprecated or obsolete in most specifications in favor of the CSS property text-decoration: underline
var (Variable) This logical inline element is used to indicate a variable (an identifier that occurs in a programming language or a mathematical expression), with any enclosed text generally rendered in italics.
kbd (Keybord Input)This inline element logically indicates text as keyboard input. A browser generally renders text enclosed by this element in a monospaced font
code (Code Listing) This element indicates that the enclosed text is source code in a programming language. Usually it is rendered in a monospaced font.
cite (Citation) This element indicates a citation from a book or other published source and usually is rendered in italics by a browser.
